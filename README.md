# ReconEye

**ReconEye** is a console-based Open Source Intelligence (OSINT) tool designed to automate data collection, verify nickname availability across multiple platforms, and gather target intelligence.

---

## 🚀 Features
*   **Nickname Verification:** Swiftly scan targeted platforms and websites for specific usernames.
*   **Intelligence Gathering:** Utilize the built-in biography module to extract and compile additional target details.
*   **Interactive CLI:** A simple, clean, and color-coded command-line interface for seamless navigation.
